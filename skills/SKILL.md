# 신규 F&B 입점 종합 검토 스킬 (entry-review v1.1)

> **통합 원본**: dev-simulation + fnb-market + field-management 3개 스킬 통합
> **신규 모듈**: 오피스 포획시장 분석 / 주변 식당가 경쟁 분석 / 구내식당(B2B) 타당성 분석

## 트리거 키워드

신규입점, 입점 검토, 입지 검토, 출점 타당성, 점포 개발, 점포 입점, 구내식당, 사내식당, 기업 카페테리아, 포획시장, 캡티브마켓, 식당가 경쟁, 임대 조건, 보증금, MG, 관리비, 손익 계산, ROI

---

## 워크플로우 개요

사용자가 후보지 관련 이미지(건물 개요, 층별 안내, 도면, 입주기업 목록 등)를 제공하면 아래 9단계를 순차 실행하여 **Excel 9시트 + PPT 10장 + HTML 인터랙티브 대시보드** 3종 세트를 생성합니다.

---

## Step 1: 이미지 분석 및 데이터 추출

사용자가 제공한 이미지를 분석하여 아래 핵심 데이터를 추출합니다.

### 추출 항목
| 카테고리 | 추출 데이터 |
|----------|------------|
| 건물 기본 | 건물명, 주소, 준공일, 총 층수, 연면적, 용도지역 |
| 대상 공간 | 위치(층·호수), 전용면적(PY/㎡), 구조, 전면 폭 |
| 임대 조건 | 보증금, 월 MG(최소보장매출), 관리비, 부가세 포함여부 |
| 입주 현황 | 입주기업 목록, 업종, 추정 직원수, 입주율 |
| MD 방향성 | 건물주/PM이 제시한 업종 방향성, 제한 업종 |
| 주변 환경 | 도로 접근성, 대중교통, 주차, 주변 시설 |

### 데이터 보완
- 이미지에서 추출할 수 없는 데이터는 웹 검색(네이버 부동산, 호갱노노, 국토교통부 실거래가 등)으로 보완
- 추정치에는 반드시 `(추정)` 표기

## Step 2: 입지·건물 기본 분석

### 분석 항목
1. **입지 등급 평가** (S/A/B/C/D)
   - 유동인구 밀집도
   - 대중교통 접근성 (지하철 도보 거리, 버스 노선 수)
   - 주변 상권 성숙도
   - 배후 수요 (오피스/주거/학교/병원)

2. **건물 등급 평가**
   - 준공 연도 및 상태
   - 건물 인지도/브랜드
   - 주차 대수 및 편의시설
   - 1층 가시성 및 접근성

3. **SWOT 분석**
   - Strengths / Weaknesses / Opportunities / Threats

---

## Step 3: 오피스 포획시장(Captive Market) 분석 ★신규

건물 및 주변 단지의 상주 직원 기반 식사 수요를 정량 분석합니다.

### 분석 공식
```
[일일 식사 수요 계산]
상주인원 × 출근율(85%) × 외식율(70%) × 건물이용율(40%) × 점유율(5~10%)
= 일일 예상 고객수

[월 매출 추정]
일일 고객수 × 객단가 × 영업일수(22일)
```

### 세부 분석
| 항목 | 분석 내용 |
|------|----------|
| 자체 건물 상주 | 해당 건물 입주기업 직원수 합산 |
| 단지 전체 상주 | 동일 단지/테크노밸리 내 전체 상주인원 |
| 입주 예정 | 분양 완료·입주 예정 기업 추가 수요 |
| 시간대 분포 | 점심(70%) / 저녁(20%) / 기타(10%) |
| 계절 변동 | 여름 휴가철·명절 감소율 반영 |

### 출력
- 포획시장 규모 (명/일, 원/월)
- 시간대별 수요 분포 차트
- 입주율 변화에 따른 수요 시나리오 (현재/50%/75%/100%)

---

## Step 4: 주변 식당가 경쟁 분석 ★신규

### 공급 부족 지수 계산
```
공급 부족 지수 = 상주인원 ÷ 주변 식당 총 좌석수

★★★★★ 매우 부족 (8명/석 이상) → 즉시 입점 권장
★★★★☆ 부족 (6~8명/석) → 입점 유리
★★★☆☆ 보통 (4~6명/석) → 차별화 필요
★★☆☆☆ 충분 (2~4명/석) → 신중 검토
★☆☆☆☆ 과잉 (2명/석 미만) → 입점 비추
```

### 분석 항목
1. **반경 500m 식당 현황** — 업종별 분류, 좌석수 추정
2. **가격대 분포** — 저가/중가/고가 비율
3. **업종 포화도** — 한식/중식/일식/양식/카페 비율
4. **공백 업종** — 수요 대비 공급이 부족한 업종 식별

---

## Step 5: 구내식당(B2B) 타당성 분석 ★신규

오피스 건물 내 구내식당(위탁급식) 또는 B2B/B2C 하이브리드 모델의 타당성을 검토합니다.

### B2B 모델 유형
| 모델 | 설명 | 마진 구조 |
|------|------|----------|
| 순수 B2B | 건물 입주사와 단체급식 계약 | 식수 × 단가, 마진 10~15% |
| B2B+B2C 하이브리드 | 구내식당 + 일반 고객 동시 운영 | B2B 안정수익 + B2C 추가수익 |
| 순수 B2C | 일반 식당으로만 운영 | 높은 마진, 높은 변동성 |

### 분석 항목
- B2B 계약 가능 식수 추정 (상주인원 × 이용률 30~50%)
- 식단가 시장 가격 (6,000~9,000원/식)
- B2B 손익분기 식수
- B2B+B2C 혼합 시 최적 비율 시뮬레이션

---

## Step 6: 교통 인프라 호재 분석

### 분석 범위
1. **현재 교통** — 지하철역, 버스정류장, 도로 접근성
2. **확정 호재** — 착공·승인된 교통 인프라 (GTX, 지하철 연장, 도로 확장 등)
3. **예정 호재** — 계획 단계 인프라

### 호재 등급
```
★★★★★ 확정, 2년 이내 개통 → 매출 20%+ 상승 기대
★★★★☆ 확정, 3~5년 내 개통 → 매출 10~20% 상승 기대
★★★☆☆ 승인, 5년+ → 장기 가치 상승
★★☆☆☆ 계획 단계 → 불확실
★☆☆☆☆ 구상 단계 → 미반영
```

## Step 7: MD 포지셔닝 전략

### 분석 기반
- Step 3 포획시장 수요 특성 (직장인 비율, 연령대, 소득 수준)
- Step 4 경쟁 공백 업종
- Step 5 B2B/B2C 비율 최적화
- 건물주/PM MD 방향성

### 출력
1. **추천 업종 TOP 3** — 업종명, 적합도 점수, 근거
2. **업종별 예상 객단가** — 점심/저녁 구분
3. **인테리어 투자 가이드** — 평당 투자금, 총 투자 예상
4. **메뉴 전략** — 점심 세트/저녁 단품 비율, 가격 포지셔닝

## Step 8: 재무 시뮬레이션 (3시나리오)

### 시나리오 설정
| 구분 | 보수적 | 기본(권장) | 낙관적 |
|------|--------|----------|--------|
| 일 고객수 | 기본의 75% | Step 3 계산값 | 기본의 125% |
| 객단가 | 기본의 90% | 시장 평균 | 기본의 110% |

### 계산 항목
```
[매출]
월 매출 = 일 고객수 × 객단가 × 영업일수(22일)

[비용]
- 식재료비: 매출 × 35%
- 인건비: 매출 × 25%
- MG(최소보장매출): 임대 계약 조건
- 관리비: 임대 계약 조건
- 기타 경비: 매출 × 5%

[손익]
- 월 순이익 = 매출 - 총비용
- BEP(손익분기점) = 고정비 ÷ (1 - 변동비율)
- 투자회수기간 = 초기투자 ÷ 월순이익
- 2년 ROI = (24개월 순이익 합 - 초기투자) ÷ 초기투자 × 100
```

### 초기투자 산정
- 보증금
- 인테리어 (평당 × 전용면적)
- 주방설비
- 초도물품
- 예비비 (총 투자의 10%)

## Step 9: HTML 인터랙티브 대시보드 생성 ★신규

pfchangs 스타일의 인터랙티브 HTML 대시보드를 생성합니다.

### 기술 스택
- **Chart.js 4.4.1** (CDN: `cdnjs.cloudflare.com/ajax/libs/Chart.js/4.4.1/chart.umd.js`)
- **SheetJS 0.18.5** (CDN: `cdnjs.cloudflare.com/ajax/libs/xlsx/0.18.5/xlsx.full.min.js`)
- 순수 HTML/CSS/JS 단일 파일 (외부 프레임워크 없음)

### 레이아웃 구조
```
┌─────────────┬──────────────────────────────────┐
│  사이드바     │  탭 바                            │
│  (264px)     │──────────────────────────────────│
│              │                                  │
│  숫자 입력    │  탭 콘텐츠 영역                    │
│  슬라이더     │  (KPI 카드 + 차트 + 테이블)        │
│              │                                  │
│  [분석 실행]  │                                  │
│  [초기화]     │                                  │
└─────────────┴──────────────────────────────────┘
```

### 사이드바 입력 (슬라이더 13개)
| 번호 | 항목 | 기본값 | 범위 |
|------|------|--------|------|
| 1 | 단지 상주인원 | Step 3 계산값 | 1,000~50,000 |
| 2 | 입주율 | 현재값% | 0~100% |
| 3 | 출근율 | 85% | 50~100% |
| 4 | 외식율 | 70% | 30~100% |
| 5 | 건물이용율 | 40% | 10~80% |
| 6 | 점유율 | 6.7% | 1~20% |
| 7 | 점심 객단가 | 시장 평균 | 5,000~30,000 |
| 8 | 저녁 객단가 | 시장 평균 | 10,000~50,000 |
| 9 | 보증금 | 계약 조건 | 0~5억 |
| 10 | 월 MG | 계약 조건 | 0~3,000만 |
| 11 | 월 관리비 | 계약 조건 | 0~1,000만 |
| 12 | 식재료비율 | 35% | 25~45% |
| 13 | 인건비율 | 25% | 15~35% |

### 탭 구성 (10개)
| 탭 | 내용 | 차트 |
|----|------|------|
| 1. 종합요약 | 종합점수, 입점권고, KPI 4개 | 레이더차트(6축), 게이지 |
| 2. 포획시장 | 상주인원, 수요계산, 시간대분포 | 도넛, 바 차트 |
| 3. 경쟁분석 | 공급부족지수, 업종분포, 가격대 | 바, 파이 차트 |
| 4. 교통인프라 | 현재/확정/예정 호재 | 타임라인, 바 차트 |
| 5. 층별MD | 추천업종, 적합도점수 | 수평바 차트 |
| 6. 재무시뮬레이션 | 3시나리오 비교, BEP | 라인, 바 차트 |
| 7. 입주기업 | 기업목록, 업종, 직원수 | 테이블, 도넛 |
| 8. 구내식당 | B2B/B2C 모델 비교 | 스택바, 라인 |
| 9. 중장기전망 | 입주율 시나리오, 5년 전망 | 라인, 영역 차트 |
| 10. 데이터관리 | Excel 내보내기/불러오기 | - |

### 인터랙션
- **"분석 실행" 버튼**: 사이드바 입력값을 읽어 모든 차트와 KPI를 재계산
- **"기본값 초기화" 링크**: 모든 슬라이더를 Step 1~8에서 산출한 기본값으로 복원
- **슬라이더 실시간 표시**: 각 슬라이더 옆에 현재값을 파란 `<em>` 태그로 표시
- **탭 전환**: `.panel` 클래스 토글로 탭 콘텐츠 전환
- **차트 12개 이상**: Chart.js `update()` 또는 재생성으로 동적 반영

## 복수 후보지 비교 기능 (최대 3개)

사용자가 2~3개 후보지를 동시에 비교 요청할 경우 아래 추가 출력물을 생성합니다.

### 작동 방식
1. 각 후보지별로 Step 1~8을 개별 실행하여 분석 데이터 산출
2. 비교 시트/탭/슬라이드에서 나란히 비교
3. 종합 점수 기반 자동 순위 산출

### Excel 추가: 시트 10 — 후보지 비교

| 비교 항목 | 후보지 A | 후보지 B | 후보지 C |
|----------|---------|---------|---------|
| 건물명/주소 | - | - | - |
| 종합 점수 (/100) | - | - | - |
| 입점 권고 등급 | - | - | - |
| 포획시장 (명/일) | - | - | - |
| 공급부족지수 (명/석) | - | - | - |
| 월 MG | - | - | - |
| 기본 시나리오 월순익 | - | - | - |
| BEP 매출 | - | - | - |
| 투자회수기간 | - | - | - |
| 2년 ROI | - | - | - |
| 교통 호재 등급 | - | - | - |
| **추천 순위** | - | - | - |

하단에 레이더 차트(6축: 입지/포획시장/경쟁/교통/재무/임대조건)로 후보지 겹쳐 비교.

### PPT 추가: 슬라이드 11 — 후보지 비교 총괄

- 3개 후보지 핵심 KPI 나란히 비교 테이블
- 레이더 차트 오버레이
- 최종 추천 순위 및 근거

### HTML 추가: 탭 11 — 후보지 비교

- 드롭다운으로 비교 대상 선택 (후보지 A/B/C)
- KPI 카드 3열 병렬 배치
- 레이더 차트 오버레이 (Chart.js datasets 3개)
- 항목별 색상 코딩: 최우수=녹색, 최하=빨강
- 하단 종합 추천 배너

## 종합 의사결정 기준

### 점수 산정 (100점 만점)
| 항목 | 배점 | 평가 기준 |
|------|------|----------|
| 입지 등급 | 15점 | S=15, A=12, B=9, C=6, D=3 |
| 포획시장 규모 | 20점 | 일 300명↑=20, 200↑=15, 100↑=10, 100↓=5 |
| 경쟁 강도 | 15점 | 공급부족지수 기반 (★5=15, ★4=12, ...) |
| 교통 호재 | 10점 | 호재 등급 기반 |
| 재무 건전성 | 25점 | 기본 시나리오 월순익·ROI 기반 |
| 임대 조건 | 15점 | MG 부담률, 렌트프리 등 |

### 권고 등급
```
90~100점: ★★★★★ 즉시 입점 강력 권장
75~89점:  ★★★★☆ 입점 권장
60~74점:  ★★★☆☆ 조건부 입점 권장 (협상 필요)
45~59점:  ★★☆☆☆ 신중 검토 (리스크 높음)
44점 이하: ★☆☆☆☆ 입점 비추
```
